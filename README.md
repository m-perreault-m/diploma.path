## What DiplomaPath does

DiplomaPath helps you:

- **Explore Ontario course prerequisites** (Grade 9 â†’ Grade 12)
- Build a multi-year **course plan** (Grade 9, 10, 11, 12)
- See what courses **unlock future options**
- **Search** by course code (e.g., `ENG4U`) or course name
- **Share** a plan with a link (URL-based sharing)
- **Print** or save your plan for later

This tool is designed to be:

- **Fast** (static site)
- **Simple** (no logins)
- **Private** (runs locally in your browser)
- **Easy to share** with students and families
